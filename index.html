INSPECTION PROTOCOL
01Maintenance checks must be completed a minimum of once per month, conducted during the night shift.
02Test each unit for blue light activation, strobe function, and call connectivity. Document results for all locations.
03Any unit found out of service must be flagged immediately. Submit a work order or help desk ticket and note the ticket number in the comments column.
04If a unit remains out of service at the next inspection, file a follow-up work order and document accordingly.
05Upon completion, scan this record and submit via email to the CSS Director and Emergency Manager. Retain a hard copy with the CSS Director.
